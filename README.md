# Network-Checker
<img width="745" height="205" alt="Screenshot 2026-03-29 221809" src="https://github.com/user-attachments/assets/20091bd0-16cd-44f9-a906-fdbe92310221" />

Features:

Live Host Detection: Scans a /24 subnet using ICMP Echo requests.

Service Validation: Checks for common service availability (e.g., HTTP Port 80).

Real-time Reporting: Categorizes devices as "SUCCESS" (Online) or "FAILED" (Offline).

User-Defined Scanning: Allows manual input of the network prefix.
